## 🗺️ インタラクティブマップの使い方

1. `ANA_ヨーロッパ旅行マップ.html` をブラウザで開く
2. 地図上の空港マーカー（各国旗アイコン）をクリック
3. 右パネルに以下が表示されます：
   - 市内主要観光スポット
   - 日帰り旅行先（電車・バス所要時間付き）
   - 旅行スタイルの特徴

> ※ 地図表示にはインターネット接続が必要です（OpenStreetMap / CARTO使用）
